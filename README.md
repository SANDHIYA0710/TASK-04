# TASK-04

## Mini Project 1: Employee Management System

This project is used to manage employee details.
Employees are stored using a list of dictionaries.
The user can add, display, update, and delete employee records through a menu-driven program.
It helps in understanding CRUD operations and data storage using Python data structures.

Concepts Used: List, Dictionary, Functions, Loop, Condition

## Mini Project 2: Student Report Card

This project generates a report card for students.
Student marks for three subjects are stored in a dictionary.
The program calculates total marks, average, and grade, and displays the report in a formatted manner.

Concepts Used: Dictionary, Functions, String Formatting, Condition

## Mini Project 3: Shopping Cart System

This project simulates a simple online shopping cart.
Products are stored in a list of dictionaries with name, price, and quantity.
Users can add items, remove items, display cart details, and calculate the total bill.

Concepts Used: List, Dictionary, Loop, Functions

## Mini Project 4: Login & User Validation

This project implements a basic authentication system.
Usernames and passwords are stored in a dictionary, and the program validates login credentials entered by the user.

Concepts Used: Dictionary, Condition

## Mini Project 5: Unique Visitor Counter

This project counts the number of unique visitors.
Visitor names are stored in a set, which automatically removes duplicate entries.
The program displays the total count of unique visitors.

Concepts Used: Set, Loop

## Mini Project 6: String Formatter Tool

This project demonstrates string formatting techniques in Python.
The program displays a formatted sentence and shows text alignment using left, right, and center padding.

Concepts Used: String Formatting

## Mini Project 7: Bank Account System 

This project simulates basic banking operations.
Account holder names and balances are stored in a dictionary.
Users can create accounts, deposit money, withdraw money, and check balance.

Concepts Used: Dictionary, Functions, Condition

## Mini Project 8: Voting System

This project counts votes for different candidates.
Votes are stored in a dictionary, and the program determines the winner based on the highest vote count.

Concepts Used: Dictionary, Loop

## Mini Project 9: Course Enrollment System

This project manages student course enrollment.
Student names are stored in a dictionary, and their courses are stored in a list.
Users can add students, update course lists, and display student details.

Concepts Used: List, Dictionary, Loop

## Mini Project 10: Number Utility Tool

This project performs number conversions and formatting.
It converts numbers to binary, octal, and hexadecimal, and formats large numbers using comma formatting and scientific notation.

Concepts Used: Functions, Formatting
